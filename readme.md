## Jira
awdesk.atlassian.net


## BackEnd

**Stack:**
1. Java 17
2. Gradle 7.2
3. Postgres
4. Spring Boot, JPA, MVC


**Initialization:**
1. Add DB in Postgres (application.yml)
2. Add Data Source in IDEA
3. Run init.sql
4. Run HelpdeskApplication Spring Boot



## Front

**Stack:**
1. Vue 3
2. Vite 3
3. Tailwind CSS
4. Flowbite


**Initialization:**
1. cd front
2. npm i
3. npm run dev
